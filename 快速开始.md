# ⚡ 快速开始（5分钟上手）

> 如果你已经会基本操作，看这个就够了

## 1️⃣ 环境检查（1分钟）

```bash
# 检查 Python（应该显示版本号）
python --version

# 检查 ADB（应该显示版本号）
adb version

# 检查设备连接（应该显示你的设备）
adb devices
```

✅ 如果三个命令都正常，跳到第3步  
❌ 如果有问题，看 `使用说明.md` 的环境准备部分

## 2️⃣ 手机设置（2分钟）

1. 开启开发者选项：设置 → 关于手机 → 连续点击版本号7次
2. 开启 USB 调试：设置 → 开发者选项 → USB 调试 ✅
3. 连接手机：USB 连接电脑，手机上点击"允许"

## 3️⃣ 获取坐标（1分钟）

```bash
# 方法1：运行坐标辅助工具
python adb_coordinate_helper.py

# 方法2：截图后手动查看
python quick_start_adb.py
# 然后打开 quick_start_screenshot.png 查看坐标
```

## 4️⃣ 开始抢购（1分钟）

```bash
python fast_click_snatch.py
```

按提示输入：
- 按钮 X 坐标：`540`（你的实际坐标）
- 按钮 Y 坐标：`1600`（你的实际坐标）
- 其他参数：直接回车用默认值

输入 `y` 确认，等待3秒后自动开始！

## 🛑 停止

按 `Ctrl + C` 停止
